## 1. Problem Statement

Visually impaired users need assistive AI that processes voice commands cognitively
and responds contextually. The goal is to build a speech-based cognitive agent to
read text, identify objects, or answer queries.

## 2. Objectives

- Convert speech → text → intent parsing
- Design action response reasoning system
- Demonstrate real-time assistive interaction

## 3. Expected Outcomes

- Build cognitive interaction loop (Listen → Process → Respond)
- Enhance accessibility using cognitive NLP

## 4. Tools Used

Python · Speech API · YOLOv3-tiny · Tesseract OCR · Python Flask · OpenCV · Watson Assistant

## 5. Code Files (in order)

1. `app.py` — Flask backend with intent parser, YOLO, OCR
2. `templates/index.html` — Accessible frontend UI
3. `requirements.txt` — Python dependencies
